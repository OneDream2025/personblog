响应式个人博客网站


### 项目背景
这是一个响应式个人博客网站的前端页面，包含首页、文章列表、文章详情、关于我等页面，使用原生技术实现所有交互效果。


### 功能需求
1. **首页**
   - 导航栏：Logo、首页、文章、关于我、搜索图标
   - 轮播图：展示3张精选文章封面，支持自动轮播（5秒间隔）和手动切换
   - 最新文章：展示最新6篇文章的卡片，包含封面图、标题、摘要、发布日期、阅读量
   - 页脚：版权信息、社交链接图标


2. **文章列表页**
   - 侧边栏：分类列表、热门标签云、推荐阅读
   - 文章列表：支持分页，每页显示10篇文章
   - 筛选功能：按分类筛选、按标签筛选
   - 搜索功能：支持按标题关键词搜索


3. **文章详情页**
   - 文章标题、作者、发布日期、阅读量、分类、标签
   - 文章正文（Markdown 渲染为 HTML）
   - 目录导航：自动提取 h2-h4 标题生成目录，点击跳转
   - 返回顶部按钮：滚动超过300px时显示
   - 相关文章推荐：同分类下的其他文章


4. **关于我页面**
   - 个人头像、昵称、简介
   - 个人技能展示：进度条形式展示技能熟练度
   - 联系方式：邮箱、GitHub、微博等


5. **通用功能**
   - 响应式布局：适配手机（<768px）、平板（768px-1024px）、桌面（>1024px）
   - 暗黑模式：支持明暗主题切换，记住用户偏好
   - 加载动画：页面加载时显示骨架屏
   - 图片懒加载：图片进入视口时才加载

## 约束类型
- **技术栈或依赖约束**：
    - 禁止使用任何前端框架（React/Vue/Angular等）
    - 禁止使用任何 CSS 预处理器（Sass/Less）
    - 禁止使用任何第三方 UI 库（Bootstrap/Element等）
    - 必须使用 ES6+ 语法
    - 必须使用 CSS3 特性（Flexbox/Grid）
    - 禁止使用内联样式

- **架构或模式约束**：
    - 必须采用模块化设计，每个功能独立文件
    - CSS 必须采用 BEM 命名规范
    - JavaScript 必须使用 ES6 Module

- **代码风格或规范约束**：
    - HTML 必须符合语义化标准
    - CSS 类名采用 BEM 规范：block__element--modifier
    - JavaScript 变量命名采用 camelCase
    - JavaScript 常量命名采用 UPPER_SNAKE_CASE
    - 所有函数必须有 JSDoc 注释

- **业务逻辑约束**：
    - 轮播图必须支持触摸滑动切换（移动端）
    - 分页必须支持首页、上一页、页码、下一页、末页
    - 搜索必须支持实时搜索（输入时即时过滤）
    - 暗黑模式切换必须有过渡动画

- **非代码回复约束**：
    - 必须附带页面结构说明文档
    - 必须附带响应式断点设计说明
    - 必须附带功能交互流程说明

### 技术栈要求
原生HTML+CSS+Javascript
